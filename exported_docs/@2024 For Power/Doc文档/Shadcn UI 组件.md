```plain
📌 shadcn/ui 组件分类速记
🔹 基础交互（表单输入类）
Button → 按钮
Input → 输入框
Textarea → 文本域
Checkbox → 复选框
Radio Group → 单选组
Switch → 开关
Slider → 滑块
Toggle → 单个切换按钮
Toggle Group → 切换按钮组
Input OTP → 验证码输入
Select → 下拉选择框
Combobox → 组合框（搜索 + 选择）
Calendar → 日历
Date Picker → 日期选择
🔹 反馈提示（提示、弹窗类）
Alert → 警告提示框
Alert Dialog → 确认弹窗（带交互）
Dialog → 对话框
Drawer → 抽屉
Sheet → 抽屉（更简版）
Popover → 弹出层
Tooltip → 悬浮提示
Hover Card → 鼠标悬浮卡片
Toast / Sonner → 全局消息提示
🔹 布局容器（展示、分区类）
Card → 卡片容器
Accordion → 手风琴
Collapsible → 可折叠区域
Tabs → 标签切换
Separator → 分割线
Sidebar → 侧边栏
Navigation Menu → 导航菜单
Menubar → 菜单栏
Breadcrumb → 面包屑导航
Scroll-area → 滚动容器
Resizable → 可调整大小的容器
🔹 数据展示（内容展示类）
Table → 表格
Data Table → 数据表格（带筛选、分页）
Pagination → 分页器
Chart → 图表
Progress → 进度条
Badge → 徽章
Skeleton → 骨架屏（加载占位）
Typography → 文字排版
🔹 头像 / 用户交互
Avatar → 头像
🔹 高级功能（扩展类）
Aspect Ratio → 宽高比容器（16:9、1:1 等）
Command → 命令面板（类似 ⌘K 快捷搜索）
Context Menu → 右键菜单
React Hook Form → 表单绑定（非 UI，辅助库）
✅ 一句话总结：
输入交互 → Button / Input / Select / Checkbox ...
反馈提示 → Alert / Dialog / Toast / Tooltip ...
布局容器 → Card / Tabs / Accordion / Sidebar ...
数据展示 → Table / Chart / Progress / Badge ...
用户相关 → Avatar
高级功能 → Command / Context Menu / Aspect Ratio ...
```

}</code></pre><p id="u45c372f7" class="ne-p"><span class="ne-text">👉</span><span class="ne-text"> 当应用切换 </span><code class="ne-code"><span class="ne-text">.dark</span></code><span class="ne-text"> 类时，Sidebar 自动换颜色。</span></p><hr id="x4VJa" class="ne-hr"><h3 id="a8c606e5"><span class="ne-text">2️⃣</span><span class="ne-text"> 使用 Tailwind 的 </span><code class="ne-code"><span class="ne-text">data-[theme]</span></code></h3><p id="u03892564" class="ne-p"><span class="ne-text">SidebarProvider 或最外层容器可以挂 </span><code class="ne-code"><span class="ne-text">data-theme</span></code><span class="ne-text">：</span></p><pre data-language="plain" id="fI3ig" class="ne-codeblock language-plain"><code>&lt;SidebarProvider data-theme=&quot;blue&quot;&gt;
  &lt;Sidebar&gt;...&lt;/Sidebar&gt;
&lt;/SidebarProvider&gt;</code></pre><pre data-language="plain" id="LYQNb" class="ne-codeblock language-plain"><code>[data-theme=&quot;blue&quot;] {

}</code></pre><p id="u2b24864c" class="ne-p"><span class="ne-text">👉</span><span class="ne-text"> 可以实现多套主题（比如 light / dark / brand-blue / brand-green）。</span></p><hr id="lNJvL" class="ne-hr"><h3 id="700b0713"><span class="ne-text">3️⃣</span><span class="ne-text"> 动态切换（JS 控制）</span></h3><pre data-language="plain" id="TECnZ" class="ne-codeblock language-plain"><code>function toggleTheme() {

class="ne-text"> 主题包含哪些方面</span></h1><p id="uc7dae481" class="ne-p"><span class="ne-text">Sidebar 的主题变量主要分为以下几类（所有以 </span><code class="ne-code"><span class="ne-text">--sidebar-*</span></code><span class="ne-ul"><li id="u7ca84752" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">--sidebar-accent</span></code><span class="ne-text"> → 悬停 / 次级背景</span></li><li id="u519841ca" class="ne-text">3️⃣</span><span class="ne-text"> 边框 &amp; 阴影</span></h3><ul class="ne-ul"><li id="udac4e12c" data-lake-index-type="0"><code class="ne-code"><span #### 官方都举了哪些 例子？ 
<details class="lake-collapse"><summary id="uca584bc5"><span class="ne-text">📌</span><span class="ne-text"> 官方举的主题例子</span></summary><h3 id="0231d252"><span class="ne-text">1️⃣</span><span class="ne-text"> 颜色（Color Tokens）</span></h3><p id="u63e2ac7d" class="ne-p"><span class="ne-text">官方在 Sidebar 样式里内置了这些变量：</span></p><pre data-language="plain" id="m05sr" class="ne-codeblock language-plain"><code>--sidebar-background: hsl(0, 0%, 98%);

--sidebar-primary: hsl(240, 5.9%, 10%);
--sidebar-primary-foreground: hsl(0, 0%, 98%);

--sidebar-accent: hsl(240, 4.8%, 95.9%);
--sidebar-accent-foreground: hsl(240, 5.9%, 10%);

--sidebar-border: hsl(240, 5.9%, 90%);
--sidebar-ring: hsl(240, 5%, 64.9%);</code></pre><p id="ua0a182c6" class="ne-p"><span class="ne-text">👉</span><span class="ne-text"> 这些变量就是 </span><strong><span class="ne-text">背景、前景、主色、强调色、边框、ring 颜色</span></strong><span class="ne-text">，是官方文档唯一明确列出来的主题项。</span></p><hr id="OLDAV" class="ne-hr"><h3 id="69799cbb"><span class="ne-text">2️⃣</span><span class="ne-text"> 尺寸（Size Tokens）</span></h3><ul class="ne-ul"><li id="ua76672e8" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">--sidebar-width</span></code></li><li id="ufd4d1cad" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">--sidebar-width-mobile</span></code></li></ul><p id="u97fdb4d2" class="ne-p"><span class="ne-text">👉</span><span class="ne-text"> Sidebar 文档提到可以通过这两个变量控制宽度（桌面端 / 移动端）。</span></p><hr id="onh2D" class="ne-hr"><h3 id="4147eeed"><span class="ne-text">3️⃣</span><span class="ne-text"> 模式切换（Theme Switching）</span></h3><p id="u8612cbb4" class="ne-p"><span class="ne-text">官方示例里展示了：</span></p><ul class="ne-ul"><li id="u39e14334" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">:root</span></code><span class="ne-text"> → 默认（Light）主题</span></li><li id="u54851b90" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">.dark</span></code><span class="ne-text"> → Dark 主题覆盖</span></li><li id="ufb98d065" data-lake-index-type="0"><code class="ne-code"><span class="ne-text">data-[theme]</span></code><span class="ne-text"> → 自定义品牌主题</span></li></ul><p id="u26a4baa9" class="ne-p"><span class="ne-text">示例代码片段：</span></p><pre data-language="plain" id="yijWB" class="ne-codeblock language-plain"><code>:root {
  --sidebar-background: hsl(0 0% 98%);
  --sidebar-foreground: hsl(240 10% 3.9%);

.dark {
  --sidebar-background: hsl(240 5.9% 10%);
  --sidebar-foreground: hsl(0 0% 98%);
}</code></pre><hr id="muBkE" class="ne-hr"><h1 id="3060c7d6"><span class="ne-text">✅</span><span class="ne-text"> 结论</span></h1><p id="u1dcf27c5" class="ne-p"><span class="ne-text">官方 </span><strong><span class="ne-text">Sidebar Theme 的例子只包含两类</span></strong><span class="ne-text">：</span></p><ol class="ne-ol"><li id="u11b0d1db" data-lake-index-type="0"><strong><span class="ne-text">颜色</span></strong><span class="ne-text">（background, foreground, primary, accent, border, ring）</span></li><li id="u8901dcc5" data-lake-index-type="0"><strong><span class="ne-text">宽度</span></strong><span class="ne-text">（sidebar-width, sidebar-width-mobile）</span></li></ol><p id="ua5cbbe0a" class="ne-p"><span class="ne-text">⚠️</span><span class="ne-text"> 官方并没有直接给出 </span><strong><span class="ne-text">字体、圆角、阴影、动画</span></strong><span class="ne-text"> 的示例，这些属于 </span><strong><span class="ne-text">扩展思路</span></strong><span class="ne-text">，开发者可以在 </span><code class="ne-code"><span class="ne-text">--sidebar-*</span></code><span class="ne-text"> 命名空间下自由加。</span></p><hr id="vmGuN" class="ne-hr"><p id="u8af673bb" class="ne-p"><span class="ne-text">要不要我帮你整理一个 </span><strong><span class="ne-text">「官方示例 + 扩展版 Design Tokens」对照表</span></strong><span class="ne-text">？这样你能一眼看出官方原生支持的主题项，和可以自定义扩展的项。</span></p><p id="u8299a310" class="ne-p"><br></p></details>
