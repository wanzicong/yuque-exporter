为什么要学python, 为什么要学多个编程语言, 大家认为有必要学习一门与工作无关的语言吗？

他们都说重要的是编程思想， 不是编程语言！！！

java 都没学好呢， 学什么python .......  



刚开始我也认为没有必要在学习一门新的编程语言， 无非是按照以前的做法把之前做的crud 用另一门语言翻译一遍， 那我为什么不直接用我现在已经熟练掌握的语言呢?





我学一门编程语言是为了 逛github 的时候能够发现更多有趣的项目和想法， 因为我感觉编程世界很大, 绝不是停留在crud ，面向数据库编程,



语言没有好坏， 不同的是他们因为语言特性 和 固定人群所形成的社区氛围，

比如java 因为强面向对象的特性和web 后端处理的强大社区和现有的产品能力【srping springboot ...】形成了几乎后台开发垄断的地位

再比如 javascript 因为灵活，不需要编译直接刻运行在浏览器端，形成了强大的web 前端开发的垄断地位， 但是随着前段项目的复杂程度逐渐变大，形成了后面的typescript 

在在比如python 由于语法简单，运行简单，入门简单，被称为是最像自然语言的编程语言，在加上大部分做算法和人和ai 的人钟爱这个编程语言形成了强大的开源社区能力， 被国内的很多营销号一顿吹捧，提供了很多关于办公啊, 语音处理啊等等的开源包， 直接安装即可使用..

所以python 目前被很多职场人士，很多非计算机行业的人群使用， 制作自己的一些工具...
