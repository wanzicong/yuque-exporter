+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">aspect-auto</font>`<font style="color:rgb(15, 17, 21);">：你爱啥样啥样，我不管。（默认）</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">aspect-square</font>`<font style="color:rgb(15, 17, 21);">：给我长宽一样，必须是</font>**<font style="color:rgb(15, 17, 21);">正方形</font>**<font style="color:rgb(15, 17, 21);">！</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">aspect-video</font>`<font style="color:rgb(15, 17, 21);">：就按</font><font style="color:rgb(15, 17, 21);"> </font>**<font style="color:rgb(15, 17, 21);">16:9</font>**<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">来，电视啥样你啥样。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">aspect-[4/3]</font>`<font style="color:rgb(15, 17, 21);">：自定义，我说几比几就几比几。（比如老式电脑屏幕 4:3）</font>

#### **<font style="color:rgb(15, 17, 21);">2. box-sizing (盒模型)</font>**
**<font style="color:rgb(15, 17, 21);">说白了：</font>**<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">规定</font>**<font style="color:rgb(15, 17, 21);">算尺寸的方法</font>**<font style="color:rgb(15, 17, 21);">。</font>

+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">box-border</font>`<font style="color:rgb(15, 17, 21);">：你写的宽度</font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">width</font>`<font style="color:rgb(15, 17, 21);">，</font>**<font style="color:rgb(15, 17, 21);">已经包含了边框和内边距</font>**<font style="color:rgb(15, 17, 21);">。这是最省心、最常用的。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">box-content</font>`<font style="color:rgb(15, 17, 21);">：你写的宽度</font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">width</font>`<font style="color:rgb(15, 17, 21);">，</font>**<font style="color:rgb(15, 17, 21);">只算内容</font>**<font style="color:rgb(15, 17, 21);">，边框和内边距额外加出去，盒子容易变大。（浏览器默认，很坑）</font>

#### **<font style="color:rgb(15, 17, 21);">3. display (显示类型)</font>**
**<font style="color:rgb(15, 17, 21);">说白了：</font>**<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">定义它是个</font>**<font style="color:rgb(15, 17, 21);">什么玩意儿</font>**<font style="color:rgb(15, 17, 21);">。这是布局的</font>**<font style="color:rgb(15, 17, 21);">根基</font>**<font style="color:rgb(15, 17, 21);">。</font>

+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">block</font>`<font style="color:rgb(15, 17, 21);">：老实人，</font>**<font style="color:rgb(15, 17, 21);">独占一行</font>**<font style="color:rgb(15, 17, 21);">。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">inline</font>`<font style="color:rgb(15, 17, 21);">：机灵鬼，跟</font>**<font style="color:rgb(15, 17, 21);">文字一样挨着排</font>**<font style="color:rgb(15, 17, 21);">。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">inline-block</font>`<font style="color:rgb(15, 17, 21);">：墙头草，对外装</font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">inline</font>`<font style="color:rgb(15, 17, 21);">，对内是</font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">block</font>`<font style="color:rgb(15, 17, 21);">。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">flex</font>`<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">/</font><font style="color:rgb(15, 17, 21);"> </font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">inline-flex</font>`<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">弹性盒</font>**<font style="color:rgb(15, 17, 21);">大佬，带着儿子们排兵布阵。（一个独占一行，一个跟着混）</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">grid</font>`<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">/</font><font style="color:rgb(15, 17, 21);"> </font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">inline-grid</font>`<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">网格</font>**<font style="color:rgb(15, 17, 21);">大佬，画棋盘布局。（一个独占一行，一个跟着混）</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">contents</font>`<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">我自己没了</font>**<font style="color:rgb(15, 17, 21);">，我的儿子们直接升级，继承我的位置。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">table</font>`<font style="color:rgb(15, 17, 21);">系列 (</font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">table</font>`<font style="color:rgb(15, 17, 21);">,</font><font style="color:rgb(15, 17, 21);"> </font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">table-row</font>`<font style="color:rgb(15, 17, 21);">,</font><font style="color:rgb(15, 17, 21);"> </font>`<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">table-cell</font>`<font style="color:rgb(15, 17, 21);">等)：把自己伪装成</font>**<font style="color:rgb(15, 17, 21);">表格</font>**<font style="color:rgb(15, 17, 21);">结构，现在很少用。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">hidden</font>`<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">隐身了</font>**<font style="color:rgb(15, 17, 21);">，直接不显示，位置也不占。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">sr-only</font>`<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">“屏幕阅读器专用”</font>**<font style="color:rgb(15, 17, 21);">，肉眼看不见，但读屏软件能念出来。给特殊人群用的。</font>
+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">not-sr-only</font>`<font style="color:rgb(15, 17, 21);">：取消上面的效果。</font>

#### **<font style="color:rgb(15, 17, 21);">4. position (定位)</font>**
**<font style="color:rgb(15, 17, 21);">说白了：</font>**<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">决定元素怎么“飘”。</font>

+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">static</font>`<font style="color:rgb(15, 17, 21);">：老实呆着，</font>**<font style="color:rgb(15, 17, 21);">别飘</font>**<font style="color:rgb(15, 17, 21);">。（默认）</font>

**<font style="color:rgb(15, 17, 21);">说白了：</font>**<font style="color:rgb(15, 17, 21);"> </font>`**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">position</font>**`**<font style="color:rgb(15, 17, 21);">的跟班</font>**<font style="color:rgb(15, 17, 21);">。老大定了怎么飘，它们决定“飘多远”。</font>

+ `<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">*-auto</font>`<font style="color:rgb(15, 17, 21);">：自动，浏览器你看着办。</font>

#### **<font style="color:rgb(15, 17, 21);">9. object-fit & object-position (对象适配与位置)</font>**
**<font style="color:rgb(15, 17, 21);">说白了：</font>**<font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">专门管</font>`**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);"><img></font>**`**<font style="color:rgb(15, 17, 21);">和</font>**`**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);"><video></font>**`<font style="color:rgb(15, 17, 21);">在框里怎么显示。</font>
