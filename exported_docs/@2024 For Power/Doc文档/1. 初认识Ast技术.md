>常见的技术框架

 ✅  spoon 框架

 📚 **Spoon 的优势**

+ 代码质量检查和规则验证
+ 代码自动重构
+ 性能优化（如添加监控、性能分析）
+ 插桩测试



 ✅  Javaparser 框架

 🎯 **JavaParser 的优势**

+ **完全保留原始格式和空白行****。**
+ **支持 Javadoc、行注释、块注释等多种注释格式****。**
+ **可视化 AST 结构，便于分析和修改代码****。**
+ **适用于大规模代码重构和静态分析。**



 ✅  Eclipse JDT   (Java Development Tools)  

 👉 **Eclipse JDT** 是 Eclipse IDE 提供的 Java 解析、修改和代码生成的工具。 

+ 提供 Java 编译器 API，可解析并操作 Java 代码。
+ 能够保留代码格式、注释和空白行。
+ 适用于 Eclipse 插件开发或大型项目。

 

✅  CodeModel 

+ 主要用于动态生成 Java 代码。
+ 自动管理 `import` 语句，支持 Javadoc 生成。
+ 适用于**代码生成器和模板引擎**。



✅  QDox  框架

 👉 [**QDox**](https://github.com/paul-hammant/qdox) 是一个轻量级 Java 解析器，可以解析 Java 源代码，并提取 Javadoc 注释。
