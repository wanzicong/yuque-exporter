# langchain 学习路线图
基础概念

核心模块



| 学习阶段 | 关键词 | 学习重点 | 实践目标 | 推荐资源 |
| --- | --- | --- | --- | --- |
| **阶段一：基础入门** | Prompt, Chain, OpenAI API | - LLM 基础原理- 安装 langchain- PromptTemplate 使用- LLMChain 结构 | ✅ 搭建 Hello-LangChain 项目✅ 实现基本问答 Chain | [LangChain 中文入门指南](https://github.com/liaokongVFX/LangChain-Chinese-Getting-Started-Guide)<br/>[LangChain 官方文档](https://docs.langchain.com/) |
| **阶段二：中级进阶** | RAG, Embedding, 向量数据库, Memory | - 使用 Chroma、FAISS 构建知识库- 创建 RetrievalQA Chain- 引入 Memory 模块 | ✅ 构建文档问答助手✅ 实现“上下文记忆”的多轮对话 | [B站实战视频](https://www.bilibili.com/video/BV1qCQRYXEhq/)<br/>[CSDN实战教程](https://blog.csdn.net/SPESEG/article/details/148532977) |
| **阶段三：高级应用** | Agent, Tool, 多任务自动化 | - 使用 LangChain Agent 调用外部 API- 自定义工具链（Tools）- 多 Agent 协作（CrewAI） | ✅ 构建具备工具调用能力的智能体✅ 打造任务自动处理系统 | [LangChain Agent 教程](https://python.langchain.com/docs/modules/agents)<br/>[CrewAI 示例](https://github.com/joaomdmoura/crewAI) |
| **阶段四：生产能力** | LangGraph, LangSmith, 部署 | - 使用 LangGraph 进行流程编排- LangSmith 日志调试与评估- 服务部署与监控 | ✅ 部署生产级对话服务✅ 搭建可视化评估系统 | [LangGraph 官网](https://www.langchain.com/langgraph)<br/>[LangSmith 官网](https://www.langchain.com/langsmith) |
| **全阶段辅助** | 实战项目, 社区资源 | - 项目驱动学习- 阅读开源代码- 社区交流和问题反馈 | ✅ 快速积累实战经验✅ 参与社区共建 | [GitHub 实战合集](https://github.com/hwchase17/langchain-hub)<br/>[Reddit / Discord 社区](https://www.langchain.com/community) |






| 功能 | 类 / 方法 | 描述 |
| --- | --- | --- |
| 模型调用 | `ChatOpenAI` / `OpenAI` / `HuggingFaceHub` | 封装对各类 LLM 的调用（如 GPT、Claude） |
| Prompt 模板 | `PromptTemplate` / `ChatPromptTemplate` | 构建输入模板并传参 |
| Chain 构建 | `LLMChain` / `SimpleSequentialChain` | 单步或多步调用语言模型 |
| Output 格式化 | `StrOutputParser` / `JsonOutputParser` | 格式化 LLM 的输出结果 |




| 功能 | 类 / 方法 | 描述 |
| --- | --- | --- |
| 文档加载 | `TextLoader`<br/>, `UnstructuredFileLoader`<br/>, `PyPDFLoader` | 加载 `.txt`<br/>、`.pdf`<br/>、`.docx`<br/> 等文档 |
| 文本分块 | `CharacterTextSplitter`<br/>, `RecursiveCharacterTextSplitter` | 将长文档切分成小段文本 |
| 文档预处理 | `Document`<br/>, `load_documents()` | 封装与清洗文档对象 |




| 功能 | 类 / 方法 | 描述 |
| --- | --- | --- |
| 向量生成 | `OpenAIEmbeddings`<br/>, `HuggingFaceEmbeddings` | 将文本转为向量 |
| 向量存储 | `Chroma`<br/>, `FAISS`<br/>, `Pinecone`<br/>, `Weaviate` | 存储和检索嵌入向量 |
| 相似搜索 | `.similarity_search()`<br/>, `.as_retriever()` | 实现知识检索（RAG）功能 |




| 功能 | 类 / 方法 | 描述 |
| --- | --- | --- |
| 上下文记忆 | `ConversationBufferMemory`<br/>, `ConversationSummaryMemory` | 保存上下文对话信息 |
| 与 Chain 集成 | `memory`<br/> 参数 | 用于多轮对话场景，提升连贯性 |




| 功能 | 类 / 方法 | 描述 |
| --- | --- | --- |
| Agent 创建 | `initialize_agent()`<br/>, `AgentExecutor` | 创建智能体，支持动态任务执行 |
| 工具封装 | `Tool`<br/>, `tool()`<br/> 装饰器 | 封装自定义函数为可调用工具 |
| 多 Agent 协同 | `CrewAI`<br/>, `LangGraph` | 构建任务分工与协同机制（需外部库） |


| 类型 | 集成类/方法 | 描述 |
| --- | --- | --- |
| 搜索类 | `SerpAPIWrapper`<br/>, `DuckDuckGoSearchResults` | 集成网络搜索能力 |
| 计算类 | `LLMMathChain`<br/>, `Calculator` | 做数学计算、单位转换等 |
| 文件类 | `CSVLoader`<br/>, `PandasDataAgent` | 加载和查询 CSV、Pandas 表格数据 |
| API 调用 | `RequestsWrapper`<br/>, `Tool.from_function()` | 调用外部 API 接口 |
| 浏览器类 | `playwright_tool`<br/>, `ArxivSearchTool` | 自动化浏览、文献搜索（社区扩展） |


| 功能 | 工具 | 描述 |
| --- | --- | --- |
| 日志与调试 | `LangSmith` | 记录链路调用过程，方便测试和回溯 |
| 工作流编排 | `LangGraph`<br/>, `LCEL` | 编排多模型任务流程（如条件判断、循环） |
| 模型评估 | `RunEvaluator` | 对模型输出进行打分与分析 |


| 场景 | 推荐组合 API |
| --- | --- |
| 文档问答系统 | `TextLoader` + `TextSplitter`+ `Chroma`<br/> + `RetrievalQA` |
| 聊天机器人 | `ChatOpenAI`<br/> + `ConversationMemory`<br/> + `LLMChain` |
| 智能客服 | `Tool`<br/> + `AgentExecutor`<br/> + `LangSmith` |
| 多轮对话 RAG | `ChatPromptTemplate`<br/> + `FAISS`<br/> + `RetrieverChain`<br/> + `Memory` |
