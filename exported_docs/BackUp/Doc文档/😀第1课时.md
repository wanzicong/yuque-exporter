# 第5课 No wrong numbers
## 文章大意
这篇文章描述了詹姆斯·斯科特先生在Silbury拥有一家汽车修理厂，**并购置了Pinhurst的另一家修理厂**。由于新修理厂无法**安装电话**，斯科特先生**购买了十二只信鸽**，建立了一种通过信鸽传递消息的私人'电话'服务。

文章提到了信鸽在**三分钟内完成Pinhurst到Silbury的首次消息传递**，并指出斯科特先生迄今为止已经通过这种方式发送了大**量备件请求和其他紧急消息**。
